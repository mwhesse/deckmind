🚀 Deckmind Server

📊 Server Status

✅ API Server: Running on port 8088
⚛️ React UI: Runs separately on port 3000 in development

🔧 Development Setup

For development with hot reloading, run client and server separately:

Terminal 1 - Start Server (API)

cd server
npm install
npm run dev

🚀 Server runs on http://localhost:8088

Terminal 2 - Start Client (React UI)

cd client
npm install
npm start

⚛️ Client runs on http://localhost:3000 with hot reloading

Terminal 3 - Build Agent Images (optional)

docker build -t deckmind/claude-agent:latest ./claude-agent
docker build -t deckmind/codex-agent:latest ./codex-agent
